¿Qué es el alojamiento web?

Es el espacio donde se aloja un sitio web para que sea visible en Internet por todo el que lo busque.

Esta técnica recibe distintos nombres:

  • Hosting web
  • Alojamiento hosting
  • Hospedaje web

¿Por qué varían tanto los precios?

Razones:

  1. La experiencia del proveedor
  2. El respaldo institucional con el que cuenta
  3. El nivel de servicio que brindará
  4. La calidad esperada del trabajo resultante
